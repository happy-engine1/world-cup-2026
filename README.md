# FIFA World Cup 2026 — 動的トーナメント対戦図

`w-tmp/fifa/READEM.md` の要件に基づくインタラクティブな対戦図サイト。

## 機能
- **横長3カラム構成**: 左にグループ A–F、中央に決勝トーナメント、右にグループ G–L。
- **グループ成績表**: 成績順（勝点→得失差→得点→TCS）でソート。
  - 初期表示: 順位 / 国旗 / 国名 / 勝点
  - 見出しクリックで**アニメーション展開** → 試合・勝・分・敗・得点・失点・得失・TCS・勝点を表示。
- **決勝トーナメント（32チーム）**:
  - 確定枠は濃色、未確定枠は薄色（イタリック）で現順位の上位国を仮表示。
  - 国（国旗＋名前）をクリックすると、その国の**勝ち上がりルート**と**対戦する可能性のある全チーム**をハイライト。
- 国旗は [flagcdn.com](https://flagcdn.com) の画像を使用。

## 技術
Next.js 16 (App Router) / React 19 / Tailwind CSS v4 / Framer Motion / TypeScript

## 開発
```bash
npm install
npm run dev    # http://localhost:3000
npm run build  # 本番ビルド
```

## データ
`src/data/groups.ts` … 48カ国・12グループのチーム情報と想定成績（固定データ）。
`src/data/bracket.ts` … 32チームのブラケット構造、順位からの解決、ハイライト算出ロジック。

> 成績は現実的な想定値の固定データ。確定グループ（A,C,E,G,I,K）は3試合終了、
> 途中グループ（B,D,F,H,J,L）は2試合終了として、確定/未確定の見せ分けを行っている。
